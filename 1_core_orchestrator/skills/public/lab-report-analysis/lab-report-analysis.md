# 化验单分析技能

## 适用场景
用户上传了化验单（血常规、生化全套、免疫检查、肝功能、肾功能、血脂、血糖等），需要识别和分析各项指标。

## 工作流程

### 1. 识别化验单类型
- 使用 `read_file` 读取上传的化验单文件
- 识别化验单类型（血常规、生化、免疫、尿常规等）
- 提取患者基本信息（如有）

### 2. 指标解读
按以下结构分析每个指标：
- **指标名称**（中英文）
- **检测值** 和 **参考范围**
- **状态**：正常 ✅ / 偏高 ⬆️ / 偏低 ⬇️
- **临床意义**：该异常值的常见原因和可能指示的情况

### 3. 综合分析
- 将异常指标综合分析，判断可能的健康趋势
- 标注需要重点关注的异常指标组合
- 提供可能的诊断方向（仅供参考）

### 4. 输出格式

```markdown
## 化验单分析报告

### 基本信息
- 化验单类型：[类型]
- 检测日期：[日期]

### 指标分析

| 指标 | 检测值 | 参考范围 | 状态 | 临床意义 |
|------|--------|----------|------|----------|
| ... | ... | ... | ... | ... |

### 异常指标汇总
[列出所有异常指标及其综合分析]

### 建议
[基于分析结果的建议]

### ⚠️ 免责声明
本分析结果由AI生成，仅供参考。最终诊断和治疗方案请咨询专业医生。
```

## 常见化验单参考范围

### 血常规
- WBC(白细胞): 4.0-10.0 × 10⁹/L
- RBC(红细胞): 男 4.0-5.5 × 10¹²/L，女 3.5-5.0 × 10¹²/L
- HGB(血红蛋白): 男 120-160 g/L，女 110-150 g/L
- PLT(血小板): 100-300 × 10⁹/L
- NEU%(中性粒细胞比例): 50-70%
- LYM%(淋巴细胞比例): 20-40%

### 生化
- ALT(谷丙转氨酶): 0-40 U/L
- AST(谷草转氨酶): 0-40 U/L
- BUN(尿素氮): 2.8-7.2 mmol/L
- Cr(肌酐): 男 44-133 μmol/L，女 70-106 μmol/L
- GLU(血糖): 3.9-6.1 mmol/L
- TC(总胆固醇): < 5.2 mmol/L
- TG(甘油三酯): < 1.7 mmol/L

## 注意事项
- 不同医院和检测设备的参考范围可能有所不同，以化验单上的参考范围为准
- 单一指标异常不一定代表疾病，需要结合临床表现综合判断
- 始终在最后添加免责声明
