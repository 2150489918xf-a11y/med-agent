# 影像咨询技能

## 适用场景
用户上传了医疗影像图片（X光、CT、MRI、超声等），需要进行专业的影像分析。

## 工作流程

### 1. 判断是否需要影像分析
- 检查用户上传的文件类型（图片格式的医疗影像）
- 确认用户的问题涉及影像解读

### 2. 委派给影像科Agent
使用 `task` 工具将影像分析任务委派给 `imaging-agent`：

```python
task(
    description="医疗影像分析",
    prompt="请分析用户上传的影像文件。文件路径: [路径]。用户问题: [问题]。请识别影像类型，分析影像内容，标注可能的异常区域。",
    subagent_type="imaging-agent"
)
```

### 3. 综合影像分析结果
收到影像科Agent的分析结果后：
- 将影像分析结果整合到对用户的回复中
- 结合化验单或其他临床信息（如有）给出综合建议
- 始终添加免责声明

### 4. 输出格式

```markdown
## 影像分析报告

### 影像信息
- 影像类型：[X光/CT/MRI/超声]
- 检查部位：[部位]

### 分析结果
[影像科Agent的详细分析结果]

### 综合建议
[结合所有已有信息的综合建议]

### ⚠️ 免责声明
本分析结果由AI生成，仅供参考。最终诊断和治疗方案请咨询专业医生。
```

## 注意事项
- 本技能需要在 **ultra模式** 下使用（需启用子Agent调度）
- 影像科Agent当前通过MCP服务接口调用外部ML工具（接口存根，待后续对接）
- 如果MCP服务不可用，影像科Agent会明确报告
