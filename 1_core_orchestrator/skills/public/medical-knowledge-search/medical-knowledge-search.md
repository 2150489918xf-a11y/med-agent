# 医疗知识检索技能

## 适用场景
主Agent通过网络搜索工具查找医疗知识，但搜索结果不够专业、不够全面，或无法回答用户的医疗问题。

## 工作流程

### 1. 先尝试自行搜索
使用 `web_search` 和 `web_fetch` 工具搜索医疗知识：
- 使用专业医学术语作为搜索关键词
- 优先搜索权威医疗来源

### 2. 判断是否需要深度检索
如果以下任一情况成立，则委派给医疗知识检索Agent：
- 搜索结果中没有权威医学论文或指南
- 涉及罕见病或最新治疗方案
- 用户指出搜索结果不够详细
- 需要多方对比验证的医疗信息

### 3. 委派给医疗知识检索Agent
使用 `task` 工具委派：

```python
task(
    description="医疗知识深度检索",
    prompt="请深度检索关于[具体问题]的专业医疗知识。已知信息: [已搜索到的信息]。需要补充: [具体需要查找的内容]。请搜索权威医疗来源。",
    subagent_type="medical-knowledge-agent"
)
```

### 4. 综合知识并回复
收到检索结果后：
- 整合所有来源的医疗知识
- 按重要性和可靠性排列
- 标注信息来源
- 区分循证医学证据和经验性建议

## 注意事项
- 本技能需要在 **ultra模式** 下使用（需启用子Agent调度）
- 优先使用自行搜索，只有在搜索结果不足时才委派给子Agent
- 始终标注信息来源的可靠性等级
